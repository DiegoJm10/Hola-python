## Instrucciones 
### Resquisitos previos:
Para que este ejercicio funcione debemos instalar primero **Python**.

1. Abrir terminal.
2. Colocar el siguente comando 

`python3 --version`

3. Colocar el siguente comando 

`sudo apt-get instal python3`
